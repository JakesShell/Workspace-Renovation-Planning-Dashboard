# Workspace Renovation Planning Dashboard

## Overview

Workspace Renovation Planning Dashboard is a browser-based facilities planning dashboard for coordinating an office renovation initiative.

This project is positioned as a recruiter-ready frontend portfolio piece. It upgrades a static renovation brief into a more practical business-facing dashboard that presents project goals, scope, phases, risks, metrics, readiness checkpoints, and value in a clearer operational format.

## Real-World Business Use Case

This project maps to practical workflows used by:

- Facilities Teams
- Workplace Operations Teams
- Project Coordinators
- Vendor And Contractor Coordination Workflows
- Frontend Developers Building Internal Dashboards

A team may need to answer questions such as:

- What is included in the renovation scope?
- What happens in each delivery phase?
- Which risks could impact timing and budget?
- What should be checked before execution begins?
- How can stakeholders review project readiness more clearly?

This dashboard is useful for facilities planning, leadership review, execution readiness tracking, and portfolio demonstration of transforming a project brief into a working operations dashboard.
